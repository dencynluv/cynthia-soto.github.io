Hackbright Individual Project

WOURDS is a shared chat notebook for you to share with someone you care. Leave enthusiastic notes, messages, quotes or any special note you'd like for one another. Apply a "heart" to your favorite note and send it to your favorites to easily find later. With the integration of Twilio's API you are instantly notified via text message when a new note is received.
